Description complète

🎨 Bitmoji Lab — une application web qui peint des emojis sur une matrice LED NeoPixel pilotée par un BBC micro:bit V2. Ouvre un seul fichier HTML dans Chrome ou Edge, branche ton micro:bit en USB, et joue avec les LEDs sur une vraie matrice 8×8 ou 16×16. Pas d'installation. Pas de compte. Pas de cloud. Tu cliques un emoji, tu presses ENVOYER, et ça brille. CE QUE VOUS OBTENEZ • 🎭 Plus de 100 emojis pré-chargés en 7 catégories — Basiques, Robots, Véhicules, Outils, Symboles, Drapeaux, Panneaux • 🖌 Pinceau Magique — 8 couleurs préréglées + sélecteur de couleur libre, clic-glisse pour peindre, Shift+glisse pour effacer • 🔲 Taille de matrice au choix — 8×8 (64 LEDs) ou 16×16 (256 LEDs) pour plus de détail • 🎞 11 démos animées — Drapeau qui flotte, Feu tricolore, Battement de cœur, Étoile tournante, Arc-en-ciel, Smiley, Barre de chargement, Feux d'artifice, Voiture de course, Stop, Œil qui cligne • 💾 Sauvegarde tes créations localement avec des noms — recharge et renvoie quand tu veux • 🔆 Curseur de luminosité 10–100 % · mode Simple pour des aplats de couleur • 🧪 Boutons de test — remplissages Rouge / Vert / Bleu / Blanc + Effacer en un clic • 📜 Journal de communication en direct avec Effacer / Copier / Exporter • 📱 Marche sur Chromebook, Windows, macOS, Linux — partout où Chrome ou Edge tourne CONTENU DU TÉLÉCHARGEMENT • index.html — l'application complète • makecode.ts — firmware micro:bit (à flasher une fois depuis MakeCode) • SETUP.md — démarrage rapide en 5 minutes + schéma de câblage • docs/ — guide illustré, aide-mémoire, FAQ, playbook Etsy • LICENSE — licence mono-utilisateur personnelle / classe PRÉREQUIS • BBC micro:bit V2 + matrice LED NeoPixel (8×8 ou 16×16, compatible WS2812B) • Câble USB data (pas charge-seule) • Chrome ou Edge sur ordinateur / Chromebook — utilise l'API Web Serial • Câblage : micro:bit P0 → DIN matrice · 3V → VCC · GND → GND POUR QUI Enseignants animant des ateliers STEM · familles en école à la maison · clubs de code après l'école · enfants qui aiment le pixel art · makers qui veulent une toile physique pour leurs emojis. CONFIDENTIALITÉ D'ABORD Rien ne quitte votre ordinateur. Pas d'analytique, pas de connexion, pas de cloud. Toutes les créations sauvegardées vivent dans le localStorage du navigateur et sur votre disque. MISES À JOUR À VIE Chaque version publiée depuis cette annonce est à vous pour toujours.
