SLUG — THE 60-SECOND ARC

Structure

 3–10 s  CONTEXT     Chrome open · USB cable · micro:bit + matrix on the desk.
10–25 s  DEMO 1      Pick emoji from grid → click SEND → matrix mirrors it live.
25–40 s  DEMO 2      Magic Paintbrush: pick color → drag on 8×8 → SEND → matrix paints it.
40–50 s  PAYOFF      Save as "Jake's Heart" → reload tomorrow → send again.
50–55 s  CTA         "Full kit + 100+ emojis + wiring guide — link below."
55–60 s  END FRAME   Emoji frozen on the matrix for the loop preview.

Full description

🎨 Bitmoji Lab — a browser web app that paints emojis onto a NeoPixel LED matrix driven by a BBC micro:bit V2. Open one HTML file in Chrome or Edge, plug in your micro:bit over USB, and light up pixel art on a real 8×8 or 16×16 LED matrix. No install. No login. No cloud. Just click an emoji, press SEND, and watch it glow. WHAT YOU GET • 🎭 100+ pre-loaded emojis across 7 categories — Basic, Robots, Vehicles, Tools, Symbols, Flags, Road Signs • 🖌 Magic Paintbrush — 8 preset colors + full custom color picker, click-drag to paint, Shift+drag to erase • 🔲 Switchable matrix size — 8×8 (64 LEDs) or 16×16 (256 LEDs) for finer detail • 🎞 11 animated demos — Waving Flag, Traffic Light, Heart Beat, Spinning Star, Rainbow Wave, Happy Face, Loading Bar, Fireworks, Racing Car, Stop Sign, Blinking Eye • 💾 Save your designs locally with custom names — reload and send again any time • 🔆 Brightness slider 10–100 % · Simple Mode toggle for flat color fills • 🧪 Test buttons — Red / Green / Blue / White fills + one-click Clear • 📜 Live Communication Log with Clear / Copy / Export • 📱 Works on Chromebooks, Windows, macOS, Linux — anywhere Chrome or Edge runs WHAT'S IN YOUR DOWNLOAD • index.html — the full web app • makecode.ts — micro:bit firmware (flash once from MakeCode) • SETUP.md — 5-minute quick-start + wiring diagram • docs/ — illustrated guide, cheat sheet, FAQ, Etsy playbook • LICENSE — personal / classroom single-user license REQUIREMENTS • BBC micro:bit V2 + NeoPixel LED matrix (8×8 or 16×16, WS2812B compatible) • USB data cable (not charge-only) • Chrome or Edge on desktop / Chromebook — uses Web Serial API • Wiring: micro:bit P0 → matrix DIN · 3V → VCC · GND → GND WHO IT'S FOR Teachers running STEM workshops · home-schooling families · after-school coding clubs · kids who love pixel art · makers who want a physical canvas for their emoji ideas. PRIVACY FIRST Nothing leaves your computer. No analytics, no logins, no cloud. All saved designs live in your browser's localStorage and on your disk. LIFETIME UPDATES Every version shipped from this listing is yours forever. Re-download the latest ZIP anytime from your Etsy Purchases page. Happy painting! 🎨✨🚀
