Etsy 60-sec Video Shoot Card Bitmoji Lab · Script A

Pre-flight (90s)

  • micro:bit V2 powered, firmware flashed (makecode.ts)
  • NeoPixel matrix wired: VCC→3V/ext · GND→GND · DIN→P0
  • External power connected if shooting 16×16
  • Chrome or Edge desktop open, index.html loaded
  • Web Serial connected — status green, picker showed CMSIS-DAP
  • Mode auto-detected: MODE:8 or MODE:16 in log
  • Brightness slider at 5% default (camera-friendly)
  • Language matches voiceover (EN / FR / AR)
  • Phone in 9:16 on tripod, front-lit
  • Matrix angled to catch lens — no hotspot
  • Desk cleared — micro:bit + matrix + cable only
  • Props on standby: USB cable, external 5V pack for 16×16
  • Captions app ready OR overlay plan
  • Script read once, under 55s
  • Phone silenced, kids/dogs warned

Framing & lighting

  • Two-shot: laptop screen left, matrix right
  • Matrix tilted ~15° toward lens — no glare
  • Key light soft, off-axis — don't wash the pixels
  • Brightness 5% on camera looks like 50% to the eye
  • If 16×16, ramp brightness only after cover frame
  • One scene per phase — don't cut every 2s
  • Switch on words, not mid-sentence
  • 1s pause on Send button
  • Smile for final 3s — loop frame

The 60-second arc — Script A

TimeSayDo / Click
0–3s "Paint pixels in your browser. Watch them glow on a real NeoPixel matrix." Close-up NeoPixel matrix wired to micro:bit V2. USB cable in frame.
3–10s "One cable, one click — Web Serial over USB." Click Connect → picker shows CMSIS-DAP → status green.
10–20s "Pick an emoji from 100+ — send it." Emoji Library → tap a face → Send. Cut to matrix lighting up.
20–30s "Or paint your own — click, drag, Shift-click to erase." Magic Paintbrush, click-drag a heart, Shift-click one pixel. Send.
30–40s "Auto-switches 8×8 or 16×16. Save designs by name. Brightness stays kind." Toggle 16×16 (MODE:16). Save as "My Heart". Slider at 5%.
40–50s "Ten built-in animations — one click and it dances." Click Rainbow Wave. Matrix rolls color. TX/RX/ACK ticks in log.
50–55s "Link below — code BITMOJI5 drops it to five dollars." Face-cam. Eye contact with lens.
55–60s "Paint • Glow • Share." Hold on glowing matrix. Logo + handle freeze. Loop frame.

After Stop

  • Watch once, eyes off script
  • Trim to 55s (iMovie / CapCut / DaVinci)
  • Burn in captions — centered below subject
  • On-screen overlays: "Web Serial · USB · Chrome" at 0:03; "100+ emoji · 8×8 / 16×16" at 0:15; "BITMOJI5 → $5" at 0:50
  • Set first frame = lit matrix close-up
  • Export MP4/H.264, < 100 MB
  • Upload · verify autoplay · phone + sound off
