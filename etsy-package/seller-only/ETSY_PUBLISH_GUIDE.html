  • 7 listing mockup PNGs rendered into etsy-package/output/
  • Listing title under 140 chars
  • Short description starts with the strongest keyword phrase
  • 13 tags filled, no dupes, no "microbit / microbit v2 / microbit matrix" stuffing

4. Video shoot checklist

  • NeoPixel matrix wired, V2 flashed, app connected (green chip)
  • In-app brightness bumped to 30–60% for the camera
  • External power pack visible in all matrix shots
  • Phone in 9:16 mode on a tripod
  • Matrix lit from the side, not straight-on (avoids washout)
  • Clip A: picking an emoji → glowing matrix
  • Clip B: painting with Magic Paintbrush
  • Clip C: SEND → close-up of the matrix lighting up
  • Clip D: Save / reopen from gallery
  • Captions burned in (Etsy autoplays muted)
