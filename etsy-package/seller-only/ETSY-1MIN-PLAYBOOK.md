# Bitmoji Lab → Etsy 1-Minute Demo Playbook

**For Etsy listing videos (60-second cap, 9:16 portrait, autoplay-muted preview).**

---

## Pre-flight checklist

- [ ] NeoPixel matrix wired (`VCC→3V/ext`, `GND→GND`, `DIN→P0`)
- [ ] External power pack in place for 64+ LEDs
- [ ] micro:bit V2 flashed with `makecode.ts`, showing ✅
- [ ] Chrome or Edge open, `index.html` loaded
- [ ] USB connected → status chip green (BBC micro:bit CMSIS-DAP)
- [ ] Matrix size (8×8 or 16×16) set correctly in the app
- [ ] Brightness raised to 30–60% for the camera
- [ ] Emoji library scrolled to a visually striking category (flags or faces)
- [ ] Phone on tripod, 9:16 portrait, side-lit
- [ ] Desk cleared — only the board, matrix, one cable in frame
- [ ] Script rehearsed once, timed under 55 s

---

## 60-second arc

```
 0–3 s   HOOK        "Watch an emoji become light in 60 seconds."
 3–10 s  CONTEXT     Open index.html → Connect → status green.
10–30 s  DEMO        Tap ❤️ → matrix glows. Tap 🏴 → flag.
30–45 s  PAYOFF      Paint your own pixels, hit SEND, watch it glow.
45–55 s  SAVE        Name it, 💾 Save, reopen from My Saved Designs.
55–60 s  CTA         "Full kit in shop — link in bio."
```

## Scene-by-scene script

### Template A — "Paint an emoji, send it, glow it"

| Time | Voice-over | Action |
|------|------------|--------|
| 0–3 s | *"What if every emoji could glow on a real LED matrix?"* | Close-up of dark matrix, ❤️ lights up red |
| 3–10 s | *"Open one HTML file in Chrome. Click Connect. Pick your micro:bit."* | Screen recording: double-click `index.html`, Connect, picker, green status |
| 10–20 s | *"100+ emojis ready to send — faces, robots, flags."* | Scroll emoji grid, tap ❤️ → SEND, tap 🌈, tap 🏴 |
| 20–35 s | *"Or paint your own, pixel by pixel."* | Expand Magic Paintbrush, pick 🚀 blue, paint a rocket, Shift-click to erase |
| 35–45 s | *"Your drawing becomes light. For real."* | Hit SEND → cut to matrix glowing the blue rocket |
| 45–55 s | *"Name it, save it, reopen it anytime."* | Name "My Rocket", 💾 Save, open My Saved Designs, click tile, rocket returns |
| 55–60 s | *"Full kit — link in description."* | Logo + handle end-frame, hold still 3 s |
