Sample Lesson: From Emoji to Pixels

Ready-to-teach 45-minute example
From Emoji to Pixels: How Pictures Become Light
45 minutes
Ages 9–14
STEAM / Digital Art / CS
Learning Objectives
1. Wire a NeoPixel LED matrix to a micro:bit safely.
2. Connect a web app to a micro:bit over USB (Web Serial).
3. Explain how an image becomes RGB pixel data.
4. Design an original 8×8 or 16×16 emoji and send it live.

Bitmoji Lab features used in this lesson

  • USB Connect Web Serial pairing with a CMSIS-DAP prompt
  • 100+ Emoji Library 7 categories: faces, robots, vehicles, tools, symbols, flags, signs
  • Magic Paintbrush 8 color presets + custom picker, click-drag, shift-erase
  • Save Designs named slots kept in the browser
  • Matrix Size switch between 8×8 (64 LEDs) and 16×16 (256 LEDs)
